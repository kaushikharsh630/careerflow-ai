# CareerFlow AI 🚀

AI-powered resume and job matching platform that generates optimized resumes, cover letters, interview Q&A, and audio coaching using Generative AI.

## Tech Stack
- FastAPI
- Streamlit
- LangChain
- Google Gemini
- Python

## Features
- ATS Resume Match Score
- Resume Optimization
- Cover Letter Generator
- Interview Preparation
- AI Voice Coaching
